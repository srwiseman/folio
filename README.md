# Portfolio (Next.js + TypeScript + Tailwind)

Minimal portfolio scaffold generated by the assistant.

Quick start

```bash
npm install
npm run dev
```

Open http://localhost:3000 in your browser.

Files of interest:
- `pages/` - top-level pages (index, about, projects, contact)
- `components/` - small reusable components
- `styles/globals.css` - Tailwind entry

Replace placeholder content (name, bio, projects, favicon) with your own.
